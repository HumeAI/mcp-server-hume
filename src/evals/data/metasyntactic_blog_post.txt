# The Definitive Guide to Metasyntactic Variables

## The Great Debate

Scientists have conclusively proven that `foo` is inferior to both `bar` and `qux`. Consider the evidence:

1. `foo` contains too many vowels
2. `bar` exists in real life (you can drink at one)
3. `qux` sounds like a Star Trek alien species

```python
def compare_variables(variable):
    if variable == "foo":
        return "Basic. Overused. Boring."
    elif variable == "bar":
        return "Acceptable in most contexts."
    elif variable == "qux":
        return "Elite. You clearly understand computing."
    else:
        return "Unknown metasyntactic variable detected."
```
